# OpenMatch — iOS app

A SwiftUI app for iOS 17+. Sources live under `OpenMatch/`. The Xcode project is generated by [XcodeGen](https://github.com/yonaskolb/XcodeGen) from `project.yml` so the repo stays small and merge-friendly.

## Setup

```bash
brew install xcodegen
make generate
open OpenMatch.xcodeproj
```

Configure the API base URL in `OpenMatch/Networking/APIClient.swift` (defaults to `http://localhost:8080` for simulator development).

## Architecture

- **DesignSystem** — colors, typography, haptics, button styles, gradient overlays.
- **Networking** — typed `APIClient` over `URLSession` with JWT refresh; WebSocket client for `/ws/v1/chat`.
- **Persistence** — Keychain wrapper, simple cached state. (Swap to SwiftData once schemas stabilize.)
- **State** — shared `AppState`, dependency container.
- **Features/** — one folder per tab (Onboarding, Swipe, Likes, Chat, Profile) plus Safety, Settings, Transparency.

## What's not in this MVP

- Push notifications (APNs) — schema is in place, wiring is not.
- Snapshot tests using `swift-snapshot-testing` — left to first contributor to add the SwiftPM dep.
- Apple Sign-In UI (requires real Apple developer team for the cert).

## Tests

```bash
make test
```

Runs unit and UI tests against the iPhone 15 simulator. Unit tests cover swipe-deck state, undo stack, photo carousel, DTO decoding. UI tests cover signup, the four-tab shell, and the swipe gesture happy path.
